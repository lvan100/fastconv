# fastconv

[English](README.md) | 中文

最快的 map 和 struct 互转库 🚀🚀🚀

## 特色

- 零 GC
- 没有偏招
- 🚀🚀🚀🚀

## 基准测试

## 编译

```
go 1.21+
```

## 安装

```
go get github.com/lvan100/fastconv
```

## 示例

```
src := map / struct / slice
dst := map / struct / slice
fastconv.Convert(src, dst)
```
